题目链接

一道比较裸的区间 dp

设状态 fl,rf_{l,r}fl,r​ 为合并到 [l,r][l,r][l,r] 时获得的最大价值，那么对于
l≤k≤rl \le k \le rl≤k≤r 有

fl,r=max⁡(fl,r,  fl,k+fk+1,r+(al+ar)⋅ak)f_{l,r} = \max\Big( f_{l,r}, \; f_{l,k} + f_{k+1,r} + (a_l + a_r) \cdot a_k \Big) fl,r​=max(fl,r​,fl,k​+fk+1,r​+(al​+ar​)⋅ak​)

枚举时按照区间 dp 的顺序，先枚举区间长度，再枚举左右端点，接着枚举区间范围内的数进行合并转移。

分离顺序的输出

题目还要求出分离顺序。我们可以在转移时：

  • 如果枚举到的区间合并方案 优于先前方案（即可以更新区间答案）
  • 那么记录下
    pl,r=kp_{l,r} = kpl,r​=k
    表示区间 [l,r][l,r][l,r] 从 kkk 位置分离。

接着：

  • 开一个 vector<int> 数组
  • 每一个 vector 存储 某一层 的分离位置
  • 然后递归扫描每一层区间，将分离位置 push_back 到对应层的 vector 中
  • 输出时把每一层的分离位置 sort 后，从小到大输出即可
